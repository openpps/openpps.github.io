Introduction

The point of the project is to create an open-source adjustable power supply with MCU (ESP32), screen, controls, USB-C, Wi-Fi, Bluetooth, reasonably wide input/output range, portability, hackability and more.
Everything at about the size of a 20,000 mAh power bank.

Core Board

  • MCU: ESP32
  • DC input: 4.5 - 32 V
  • DC output: 1 - 25 V
  • power: up to 5A/100W
  • efficiency: up to 97%
  • CC/CV, Power Good, Temperature Protection, Overvoltage Protection.

Buck-Boost Converter IC with MCU control, ADC measurements (12 bit) and current limiting.
Core Board can also be used as a separate IoT device where you need to control/monitor DC voltage and current over Wi-Fi (lights, motors, battery charging etc.) and read various sensor data. Or as a module in other projects.

Currently in developing/prototyping stage.

Input and Output Boards

Input Board is an optional part which offers various power input options for Core Board and internal battery charging.
Can contain anything including USB-C, micro-USB, car plug adapter, solar panel input, DC jack etc.
Or even an AC-DC switched-mode power supply.
Output Board is also optional and configurable. Contain screen, controls, USB charging ports, power supply output, MCU pins etc.
USB-C and DC outputs are capable of charging notebooks.
