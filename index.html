Open Programmable Power Supply

The point of the project is to create a portable adjustable power supply with MCU (ESP32), screen, controls, USB-C, Wi-Fi, Bluetooth, reasonably wide input and output range, modular structure, hackability and more.
Everything at the size of a 20,000 mAh power bank.

Core Board

  • MCU: ESP32
  • input: 4.5 - 32 V
  • output: 0.5 - 25 V
  • power: up to 5A (100W)
  • efficiency: up to 97%
  • CC/CV, Power Good, Temperature Protection, Overvoltage Protection.

Buck-Boost Converter IC with MCU control, ADC measurements (12 bit) and current limiting. Currently in developing/prototyping stage.
Core Board can also be used as an separate IoT device where you need to control current or voltage over Wi-Fi and read various sensor values. Or as a module in other projects.

Input and Output Boards

Input Board is an optional part which offers various power input options for Core Board.
Can contain anything including USB-C, micro-USB, car plug adapter, solar panel input, DC jack etc.
Or even an AC-DC SMPS if you like.
Output Board is also optional and configurable. Can contain screen, controls, USB-A/USB-C charging ports, power supply output etc.
